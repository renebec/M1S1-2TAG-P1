Módulo 1 - Submódulo 1

Parcial 1
Calificación (CLF), Asistencia (AST) y Semáforo de Conducta (CND).




 Gráfica interactiva: Cada esfera representa un alumno. Usa tu mouse para girar, alejar, acercar y explorar la gráfica. Pasa el cursor por cada esfera para desplegar la información por alumno.

Eje x (CLF): calificación
Eje y (AST): asistencia
Eje z (CND): semáforo de conducta

El color de cada esfera es proporcional a la CALIFICACIÓN (real) (0-100); el tamaño representa el nivel de ASISTENCIA y la altura corresponde a la CONDUCTA que oscila alrededor de tres niveles: 0 (MALA), 50 (REGULAR) y 100 (BUENA), mismos que indican el nivel de atención y participación observado en el alumno durante las sesiones de aprendizaje.
